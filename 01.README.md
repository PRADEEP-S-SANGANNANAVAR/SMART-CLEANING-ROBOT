# SMART-CLEANING-ROBOT
About the Project
The Smart Cleaning Robot is an automatic floor cleaning system designed to reduce human effort. It is capable of performing multiple cleaning tasks such as sweeping, vacuum cleaning, wet mopping, and drying in a single system.
The robot uses sensors to detect obstacles and move safely without collisions. It is useful in homes, offices, hospitals, and other indoor environments where regular cleaning is required.

Why this Project
Manual cleaning requires a lot of time and effort, especially in large areas. This project provides a simple and cost effective solution by automating the cleaning process. It helps save time, reduces physical work, and ensures better cleaning.

How it Works
The robot is powered by rechargeable batteries. When switched on, the sensors start detecting obstacles around the robot.
Ultrasonic sensors detect objects in front and back, while IR sensors detect obstacles on the sides. Based on this information, the robot moves in a safe direction.
At the same time, rotating brushes collect dust and move it toward the vacuum unit. The vacuum stores dust in a dustbin. A water pump sprays water for wet cleaning, and the wiping system cleans and dries the floor.

System Flow
Start the system
Initialize sensors and components
Move forward
Detect obstacles
Change direction if needed
Perform cleaning operations
Continue the process
Stop when cleaning is complete

Code Setup
To run this project, install Arduino IDE and connect your Arduino Uno to your computer.
Open the project code in Arduino IDE and select the correct board and port. Install any required libraries if needed.
Upload the code to the Arduino. Once uploaded, power the system and the robot will start working automatically.
Make sure all hardware connections are done correctly before running the system.

Results
The robot successfully detects obstacles and avoids collisions. It performs cleaning effectively using both dry and wet methods. The system runs continuously and reduces manual effort.
